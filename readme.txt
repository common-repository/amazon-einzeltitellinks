Dieses Plugin ermöglicht die einfache Integration von Amazon.de Einzeltitellinks mit Text und Grafik. 

== Description ==

Mit diesem Plugin kannst du Einzeltitellinks von Amazon.de schnell und einfach in deinen Wordpress Artikel integrieren. Der Amazon-Einzeltitel wird mit Text und Grafik mittels iframe integriert, sodass du die Artikelbilder nicht von Amazon downloaden musst. Das Plugin ermöglicht die Eingabe deiner Partner ID in den Einstellungen, außerdem kannst du bei jeden Amazon Artikel den du einfügst bestimmen wie der nächste Text sich verhalten soll: Textfluss: left, right oder kein Textfluss.

Demo: http://www.diewebmaster.it/amazon-einzeltitellinks-plugin-wordpress/

== Installation ==

1. Hier das Amazon Wordpress Plugin herunterladen. 
2. Das so erhaltene Zip-Archiv lokal entpacken und die enthaltene Datei: amazon-einzeltitellinks.php in den Wordpress Plugin Ordner hochladen. 
3. Das Plugin aktivieren und unter Einstellungen deine Amazon Partner_ID eingeben und wenn gewünscht konfigurieren.
